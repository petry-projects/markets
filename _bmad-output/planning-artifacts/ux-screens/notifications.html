Notification Type Reference

🧺 Customer Notifications
Vendor Check-In — followed vendor arrives
Vendor Checkout — followed vendor leaves
Exception Alert — running late, sold out, not attending
Alternative Suggestion — similar vendor when favorite absent
Market Announcement — closures, special events
Market Day Reminder — morning of followed market days
🚜 Vendor Notifications
Check-In Reminder — market day morning prompt
Auto-Checkout — when market closes
Join Request Approved — market accepted you
Join Request Declined — includes manager's reason
Confirmation Request — manager asks to confirm attendance
Manager Broadcast — operational updates from manager
Follower Milestone — 10, 25, 50, 100 followers
👩‍🌾 Manager Notifications
Vendor Exception — running late, not attending
Possible No-Show — unchecked-in after market opens
New Join Request — vendor wants to join
Pre-Market Summary — confirmed/pending vendor counts
Market Close Summary — attendance recap
Co-Manager Accepted — invite accepted
Vendor Roster Change — vendor cancels a date
