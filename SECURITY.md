# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please send an email to the repository maintainers or use
[GitHub's private vulnerability reporting](https://github.com/petry-projects/markets/security/advisories/new).

We will acknowledge receipt within 72 hours and aim to provide a fix or mitigation plan within 30 days.

## Disclosure Policy

We follow coordinated disclosure. We ask that you give us a reasonable amount of time to address the issue before making any information public.
